# AMT
Apple's Medical Thing

Usefull for FTXs or roleplay purposes

Zeus module to instantly spawn patients of various presets
Zeus module to add injuries to already exsiting units (or players)
possibility to change what type of units are spawned via CBA setting

don't use the spawn module on vehicles (or do and laugh at the result)


(make sure ace_medical_statemachine_AIUnconsciousness is on otherwise you cannot use AI as training dummies)
